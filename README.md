# codealpha_Basic-chatbot
 # Project Overview

A Basic Chatbot is a simple Python program that interacts with users through text-based conversations. It responds to predefined questions and provides appropriate answers based on user input.

# Objectives
Understand how chatbots work.
Learn Python conditional statements and loops.
Practice handling user input and output.
# Technologies Used
Python 3
# Features
Greets the user.
Responds to common questions.
Handles unknown inputs.
Continues conversation until the user exits.
